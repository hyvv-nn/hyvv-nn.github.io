P1 — 저비용 임베디드 자율수거로봇

웹 2D → Webots 3D → 실물 로봇을 실시간 동기화한 3계층 검증으로 완성한 저비용 임베디드 자율수거 시제품. 실로봇 시스템과 하드웨어 통합 역량을 보여준다.

→ github.com/hyvv-nn/autoproject

P2 — 강화학습 vs 고전제어 관절 서보

1자유도 관절에서 고전 PID와 학습 PPO를 같은 조건에서 공정 비교해, 선형이면 PID·비선형이면 PPO가 목표값 수렴에 더 유리하다는 제어기 선택 경계를 수치로 제시했다.

→ github.com/hyvv-nn/autocontrol-research

P3 — ROBOTIS OP3 공차기 강화학습

MuJoCo 환경에서 ROBOTIS OP3의 공차기를 고전제어와 강화학습으로 공정 비교하고, 강화학습 기반 킥을 OP3 물리적 한계의 93%(공 속도 1.86 m/s)까지 끌어올렸다. OH! GYM의 핵심인 시뮬레이터 휴머노이드 강화학습을 직접 확보한 프로젝트.

→ github.com/hyvv-nn/op3-kick-rl
